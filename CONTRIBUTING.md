# Contributing to SentinelAI

Thank you for your interest in contributing! Please read these guidelines before submitting.

---

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/SentinelAI.git`
3. Create a virtual environment and install dependencies:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```
4. Create a feature branch: `git checkout -b feature/your-feature-name`

---

## Code Style

- Follow **PEP 8** for all Python code
- Use type hints on all public functions and methods
- Keep functions focused — one responsibility per function
- Document public APIs with docstrings

---

## Submitting Changes

1. Ensure your changes don't break existing functionality
2. Write clear, descriptive commit messages
3. Push your branch and open a Pull Request against `main`
4. Fill out the PR template completely

---

## Feature Requests

Be respectful and constructive. Harassment of any kind will not be tolerated.
